Share Button Directive

Overview

[shareButton] is a directive that converts its host element to a share button.

This directive comes with ngx-sharebuttons package.
Installation

Install the following packages

NPM

>
Import the module

The most basic share button needs only the name of the social network. e.g: shareButton="twitter"

By default, the package does not import any icon from FontAwesome, if you wish to use them, See the Icons Guide.
