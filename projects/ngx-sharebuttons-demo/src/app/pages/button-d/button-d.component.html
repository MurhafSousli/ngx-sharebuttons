Share Button Directive

Overview

[shareButton] is a directive that transforms an element into a share button.

This directive is available through the ngx-sharebuttons package.
Installation

Install the following packages

NPM

Basic Usage

The most basic share button needs only the name of the social network. e.g: shareButton="x"
