Global Options

Setting custom options

To avoid setting the same options on each share component/directive, you can set your custom options that applies on all share buttons components across your app.

Example

If you want to set the options globally while lazy loading the library, you can provide your custom config as SHARE_BUTTONS_CONFIG value, this will set the defaults before the library is loaded.
