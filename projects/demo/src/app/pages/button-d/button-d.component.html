Share Button Directive

Overview

[shareButton] is a directive that converts its host element to a share button.

This directive comes with @ngx-share/core package. Check Core Package Compodoc.
Installation

Install the following packages

NPM

YARN

Import the module

The most basic share button needs only the name of the social network. e.g: shareButton="twitter"

You can get the official colors and icons by injecting ShareButtons service in your component

By default, the core package does not load any icon from FontAwesome, if you wish to use them, you should import them manually. Follow the Icons Guide.
